# sentinel-scan-ai-
AI-powered web vulnerability scanner built using Python and Flask

Sentinel Scan AI is an AI-powered web vulnerability scanning system developed using Python and Flask. The system is designed to identify common web vulnerabilities and provide human-readable security explanations.

## Features

- Web vulnerability scanning
- SQL Injection detection
- Cross-Site Scripting (XSS) detection
- Passive security analysis
- AI-generated vulnerability explanations
- Security scoring system
- PDF report generation
- Flask-based dashboard

## Technologies Used

- Python
- Flask
- HTML/CSS/JavaScript
- BeautifulSoup
- Requests
- FPDF

## System Modules

- Crawler Module
- Active Scanner
- Passive Scanner
- AI Explanation Module
- Report Generation Module
